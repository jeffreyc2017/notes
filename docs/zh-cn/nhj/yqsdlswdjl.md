# 《瑜伽师地论声闻地讲录》

1980 年，南師懷瑾先生於十方書院開設講解《瑜伽師地論》的課程。

一个真正学佛的人有四本重要的佛学论著要读

- 《大智度论》
- 《瑜伽师地论》
- 《摩诃止观》智者大师
- 《宗镜录》永明寿禅师

佛学的学术思想分成四个范围：

- 大乘经典

  - 龙树菩萨的般若佛学系统，属于前期佛学
  - 无著、世亲兄弟菩萨的唯识法相系统，属于后期佛学

    - 《瑜伽师地论》
    - 阿底峡尊者《菩提道炬论》
    - 西藏密宗黄教宗喀巴大师《菩提道次第广论》

- 小乘经典
